# 💻 Portfolio 

Ce dépôt contient le code source de mon portfolio .<br>
Il met en avant mes projets, mes compétences techniques et les technologies que je maîtrise.

## 🎯 Objectif du projet

Ce portfolio a été conçu pour :

* Présenter mes projets académiques et personnels
* Mettre en valeur mes compétences en développement web
* Démontrer ma capacité à concevoir une interface moderne et responsive
* Servir de support dans le cadre de ma recherche de stage

---

## 🚀 Fonctionnalités

* Interface responsive (desktop / tablette / mobile)
* Navigation fluide et dynamique
* Système de filtres interactifs pour trier les projets
* Mode clair / sombre
* Organisation des projets par catégories :

  * Web
  * Jeux
  * Sécurité
  * Hackathon
  * Data

---

## 🛠️ Stack technique

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla)**
* Hébergement : **GitHub Pages**
